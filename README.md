# SamjhoAI

SamjhoAI is a AI powered copilot for 1092 citizen helpline systems.  
It helps helpline agents understand citizen complaints in different Indian languages and provides real time AI assistance during calls.

The platform can detect citizen emotions, identify distress situations, understand complaint categories and suggest proper responses instantly.

SamjhoAI also generates automated ticket drafts with department routing, priority level, SLA estimation and action items.

The system supports multilingual conversations including Hindi, Kannada, English and Hinglish. It also includes speech recognition for voice to text interaction.

Main features of the platform:

- real time transcript analysis
- multilingual support
- emotion and sentiment detection
- AI suggested agent replies
- automated ticket generation
- issue categorization
- department recommendation
- speech recognition
- live analytics dashboard
- citizen verification prompts

The interface is designed like a real government command center dashboard for helpline operators and public grievance systems.

Tech stack used in the project:

- HTML
- CSS
- JavaScript
- React
- Claude AI API
- Web Speech API
